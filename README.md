### Baghchal
Baghchal game.  
Dependency:
```
Python 3.12.4
Pip 24.2
Pygame 2.6.1
```

#### Requirement
```bash
pip install -r requirement.txt
```

#### Run
```bash
python Main.py
```
All the inputs are mouse based with only left click available.
